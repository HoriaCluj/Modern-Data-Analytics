Modern Data Analytics - Group 1

Comprehensive Datasets

🚲 Bike Traffic

Raw 15-minute EcoCounter data providing the core response variable.

👥 Demographics

Population, age, and gender from Statbel (2019-2024).

⛅ Weather

Hourly open-meteo API data (temp, rain, wind, snow) by coordinate.

⚠️ Accidents & Infra

Flanders road accidents + ECF infrastructure deficit metrics.

Methodology & Engineering

  • ⚡

    Spatiotemporal Features: Extracted season, day type, and time of day from timestamps.

  • 🔗

    Data Integration: Merged weather variables and demographic metrics per municipality.

  • ⚖️

    Severity Weighting: Accidents weighted by severity (fatal=5, serious=3, minor=1).

Traffic Prediction Model

Built using XGBoost / LightGBM regression.

Why Gradient Boosting?

  • Handles nonlinear relationships inherently
  • Captures complex feature interactions (weather × time)
  • Industry standard for tabular spatiotemporal data

MCPI Formula

Municipality Cycling Priority Index

MCPIm = α(Safety Risk) + β(Infra Deficit) + γ(Demand Density)

α Weight for Accidents
β Weight for Infrastructure
γ Weight for Traffic/Pop
